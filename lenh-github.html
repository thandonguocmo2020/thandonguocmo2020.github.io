Các lệnh làm việc với git


..


I. config 


Cấu hình 


git config --global user.name "name github"

git config --global user.email "email github"



II. Kết nối tới kho trên git tới máy



1. Thêm một kho kết nối tới git trên máy...


git remote add "Link kho"



2. Thêm dữ liệu  trong project vào bộ nhớ git


git add * 


3. Đẩy dữ liệu vào git chuẩn bị thêm vào kho


git commit -m "tin nhắn thông tin trước khi đẩy"


4. Đẩy dữ liệu lên kho


git push origin master 


<master> có thể là tên nhánh.



III. Trong quá trình làm việc có thể project phát sinh lỗi trưa kịp fix tạo ra nhánh mới để đẩy tạm code lên

 

1. git branch iss53 


<iss53> là tên nhánh.


2. Di chuyển sang nhánh để đẩy code vào


git checkout iss53


3. add file vào bộ nhớ nhánh


git add *


4. Thêm tin nhắn vào chuẩn bị đẩy


git commit -m "add fix iss53"


5. Đẩy lên nhánh


git push origin iss53



IV. Di chuyển giữa các project



1. Đầu tiên gõ lệnh


git remote -v
